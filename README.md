# Grocer Oasis Delight

A modern web application for grocery shopping built with:

- Vite
- TypeScript
- React
- shadcn-ui
- Tailwind CSS

## Development

Requirements:
- Node.js & npm - [install with nvm](https://github.com/nvm-sh/nvm#installing-and-updating)

```sh
# Install dependencies
npm install

# Start dev server
npm run dev
```
